| 기능           | API                                   | 설명              |
| ------------ | ------------------------------------- | --------------- |
| 회원가입         | `POST /api/auth/register`             | 사용자 계정 생성       |
| 로그인          | `POST /api/auth/login`                | 로그인 후 토큰 발급     |
| 빠른면접 피드백     | `POST /api/interviews/quick`          | 친절 페르소나로 바로 피드백 |
| 기본면접 피드백     | `POST /api/interviews/basic`          | 친절 페르소나 자동 선택   |
| 실전면접 피드백     | `POST /api/interviews/real`           | 친절+까칠+압박 3명 피드백 |
| 면접기록 조회      | `GET /api/interviews/history`         | 내가 했던 면접 목록 조회  |
| 면접 분석 조회  | `GET /api/interviews/recent/analysis` | 최근 면접 분석 결과 조회  |
| AI 피드백 요약 조회 | `GET /api/interviews/recent/summary`  | 최근 피드백 요약 조회    |
| 내 정보 조회 | `GET /api/auth/me` | 이름·이메일·가입일(계정정보) |
| 상세정보 조회 | `GET /api/auth/profile` | 나이·출신대학·취미·특기 |
| 상세정보 저장 | `PATCH /api/auth/profile` | 나이·출신대학·취미·특기 저장 (DB 컬럼 필요) |
| 회원탈퇴 | `DELETE /api/auth/account` | 면접 기록 삭제 후 계정 삭제 |

----------구현완료-------------------

| API                    | 역할                |
| ---------------------- | ----------------- |
| `POST /quick`          | 빠른면접 결과 저장        |
| `POST /basic`          | 기본면접 결과 저장        |
| `POST /real`           | 실전면접 결과 저장        |
| `POST /quick/audio`    | 빠른면접 음성 텍스트 결과 저장 |
| `POST /basic/audio`    | 기본면접 음성 텍스트 결과 저장 |
| `GET /history`         | 내 면접 기록 목록        |
| `GET /:interviewId`    | 특정 면접 상세 조회       | O
| `DELETE /:interviewId` | 특정 면접 삭제          |
| `GET /recent/analysis` | 가장 최근 면접 분석       |
| `GET /recent/summary`  | 대시보드용 최근 요약       |
